#  Table of Contents

Welcome to the PurpleForge lab environment.  
Below is a categorized overview of the main lab components and documentation folders.

---

##  [01_Lab_Architecture](./01_Lab_Architecture/)
Contains all technical information about the lab setup:  
VM roles, network design, VLANs, and firewall rule documentation.

---

##  [02_Scenarios](./02_Scenarios/)
Documented attack & defense use cases (e.g. ransomware, phishing, C2).  
Each scenario is mapped to detection and response paths in the lab.

---

##  [03_Automation_And_SOAR](./03_Automation_And_SOAR/)
Describes SOAR integrations like Shuffle, workflows, response triggers,  
and custom automations interacting with detection tooling.

KQL queries, Sigma rules, Sysmon configs and other detection engineering

---

##  [05_Threat_Intel_And_Enrichment](./05_Threat_Intel_And_Enrichment/)
MISP setup, CTI workflows, IOC enrichment and correlation logic  
for enhancing response quality using threat intelligence.

---

##  [06_IR_Processes_And_CTH](./06_IR_Processes_And_CTH/)
Incident Response playbooks, Chain of Custody examples,  
and practical Cyber Threat Hunting documentation.

---

##  [07_Lab_Utilities](./07_Lab_Utilities/)
Scripts and helpers for generating test data, simulating attacks,

##  [09_LLM_Integration](./09_LLM_Integration/)

##  [20_Docs_And_Presentations](./20_Docs_And_Presentations/)
Visual documentation like architecture diagrams, reports,
